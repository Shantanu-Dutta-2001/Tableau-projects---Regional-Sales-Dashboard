
# Regional Sales Dashboard

A simple Regional Sales Dashboard made in Tableau For the supermarket Sales Data.



## Features

- Simple User Interface(UI)
- Easy Navigation
- Easy to filter data
- Filter by Region-wise,Sales-wise etc
 





## Technology Stack :

- Supermarket sales data (CSV File)
- Tableau
